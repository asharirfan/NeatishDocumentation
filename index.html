Theme Installation

In start you must have a working version of WordPress already installed. For information in regard to installing the WordPress CMS, please see the WordPress Codex Installing WordPress

Once you have a working of version of WordPress, you need to download All Files and Documentation from themeforest and extract the downloaded zip to get various files including neatish.zip.

Step 1: Install Theme Via WordPress Dashboard

  • Go to 'Appearance > Themes' section
  • Click 'Add New' and select the 'Upload Theme' option
  • Choose the neatish.zip file and press 'Install Now'
  • Once the theme is uploaded you need to activate it.

If you face any problem during upload through dashboard, please upload the theme using FTP as guided below.

Step 1(Alternative): Install Theme Via FTP

  • Access your hosting server using an ftp client like FileZilla
  • Go to the 'wp-content/themes' folder of your WordPress installation
  • Extract the neatish.zip file and put the neatish folder in 'wp-content/themes/' folder
  • Go to 'WordPress Dashboard > Appearance > Themes' section to activate the theme

Set Menu

At this point if you take a look at your website you will find it is somewhat messy on the front-end. Your header will be out of order. So, the next step is the configuration of header menu.

Configuring Header Menu

  • Go to Appearance > Menus
  • Here you will be within the Edit Menus tab
  • Click on create a new menu
  • Enter the name of your menu in Menu Name
  • Then select Create Menu and your new menu will be created
  • Next scroll down the page and you’ll find Menu Settings
  • Below it there are certain checkboxes
  • Enable the Primary Menu
  • In the end click Save Menu button
  • This way your header menu gets configured

Now preview your website on the front-end and your header menu is configured.

Neatish Options

In Neatish, we have introduced a complete panel in WordPress Customizer for “Neatish Options” from where you are going to configure all the parts of this theme. In other words you can say that this part is the heart of Neatish. You can access this panel from the Customizer of Neatish.


Here you will find various sections which can be used to configure various parts of the theme. These section are for the:

  • General Settings
  • Post Settings
  • 404 Page Settings
  • About Page
  • Social Links

General Settings

In the “Neatish Options” panel the first section allows you to configure some of the “General Settings” of Neatish. You can access this section inside the customizer. Follow this path:

Dashboard > Appearance > Customize > Neatish Options > General Settings

When you click on “General Settings”, a new section appears. There are several general options inside this section.

There are four options available to configure. These options are:

  • Show Author?
  • Show Share Buttons?
  • Show Post Date?
  • Footer Text

Show Author?

Select to show author at home page. It is enabled by default.

Show Share Buttons?

Select to show share buttons at home and post page. You can share your posts on Facebook, Twitter and Google+. This option is also enabled by default.

Show Post Date?

Select to show date of the post at home page. This option is also enabled by default.

Footer Text

This is option is used to display text in the footer area.

Once you’re done, click the Save & Publish button.

Post Settings

Next section in the “Neatish Options” is for the “Post Settings”. To access this section in the customizer, follow this path:

Dashboard > Appearance > Customize > Neatish Options > Post Settings

This section contains all customizing options of the posts of Neatish.

There are three options available to configure. These options are:

  • Show Categories?
  • Show Tags?
  • Show Author box?

Show Categories?

Select to show categories at the end of the post. It is enabled by default.

Show Tags?

Select to show tags at the end of the post. This option is also enabled by default.

Show Author box?

Select to show author box at the end of the post. This option is also enabled by default.

Once you’re done, click the Save & Publish button.

404 Page Settings

To configure the settings of “404 Page Settings” section, once again go to the panel of "Neatish Options" inside the customizer. Follow this path:

Dashboard > Appearance > Customize > Neatish Options > 404 Page Settings

Your 404 page can be configured from this section.

There are two options available to configure. These options are:

  • Page Title
  • 404 Image

Page Title

This option is used to display the title of 404 page.

404 Image

This option is used to display an image on the 404 page of neatish. Click on Select Image to upload/select the image that you want to display.

Once you’re done, click the Save & Publish button.

About Page

In the "Neatish Options" panel, next section allows you to configure the About Page of Neatish. We have made an about page template. You can use this template by creating a new page.

Step 1: Creating about page

Follow this path: Dashboard > Pages > Add New

Set the “Template” to “About Page” in Page Attributes at right side of the editor and publish the page.

Now you can configure your about page from “About Page” section in “Neatish Options” panel. To access this section, go to the customizer. Follow this path:

Dashboard > Appearance > Customize > Writable Options > Contact Page

There are a number of options available to configure in this section. These options are:

  • Show Experience?
  • Experience Block
  • Show Education?
  • Education Block
  • Show Proficiency?
  • Proficiency Block
  • Show Knowledge?
  • Knowledge Block
  • Show Skillset
  • Skillset Block
  • Show Hobbies?
  • Hobbies Block
  • Show CTA Section?
  • CTA Title
  • CTA Link
  • Show CTA Button?
  • CTA Button Title
  • CTA Button Link

Show Experience?

This option is used to display experience block on your about page. Select "Yes" to display the block and then enter your experiences in the given area.

Show Education?

This option is used to display education block on your about page. Select "Yes" to display the block and then enter your education in the given area.

Show Proficiency?

This option is used to display proficiency block on your about page. Select "Yes" to display the block and then enter your proficiencies in the given area.

Show Knowledge?

This option is used to display knowledge block on your about page. Select "Yes" to display the block and then enter your knowledge in the given area.

Show Skillset?

This option is used to display skillset block on your about page. Select "Yes" to display the block and then enter your skillset in the given area.

Show Hobbies?

This option is used to display hobbies block on your about page. Select "Yes" to display the block and then enter your hobbies in the given area.

Show CTA Section?

This option is used to display call to action section on your about page. Select "Yes" to display the section and then enter the title and link for the section in the corresponding areas.

Show CTA Button?

This option is used to display call to action button on your about page. Select "Yes" to display the button and then enter the title and link of the button in the corresponding areas.

Once you’re done, click the Save & Publish button.

Social Links

The Social Links section in Neatish allows to add links of your social profiles. To access this section, go to the customizer and then click “Neatish Options”. You will find the section displayed as Social Links. Follow this path:

Dashboard > Appearance > Customize > Neatish Options > Social Links

These social links will be displayed in the “Author Box” and footer of Neatish. You can add the links of the following social profiles:

  • Facebook
  • Twitter
  • Google Plus
  • Instagram
  • LinkedIn
  • Dribble

How can I access the customizer?

In Neatish, we have used customizer to build theme options. There are two ways to access customizer.

  • Through dashboard
  • Through front-end

1. Through dashboard

Follow these steps: Dashboard > Appearance > Customize

2. Through front-end

Follow these steps:

  • Go to the front-end of your website
  • There will be an admin bar at the top
  • Click “Customize”

Both of these steps will lead you to the customizer of your theme.
